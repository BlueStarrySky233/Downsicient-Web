Installation Guide 安装指南

Step-by-Step Guide 分步指南

  1. Install a userscript manager: 安装用户脚本管理器：
    • Tampermonkey (Chromium / Gecko)
    • Other userscript managers may or may not work as intended. You may try though. 其他用户脚本管理器可能无法按预期工作。不过，您可以尝试一下。
  2. Download the script: 下载脚本：

    Click the button above for the latest version, or choose from the list. The manager will prompt to install. 点击上方按钮获取最新版本，或从列表中选择。 管理器将提示安装。

  3. Confirm installation: 确认安装：

    Click 'Install' / 'Confirm' in your extension manager. Make sure to refresh your target page. 在扩展程序管理器中点击“安装”/“确认”。请确保刷新目标页面。

  4. You are all set: 一切就绪：

    Welcome to the world of Privacy, Freedom, and Secure! You will no longer be in suffer. 欢迎来到隐私、自由和安全的世界！您将不再遭受折磨。

Troubleshooting 故障排除

  • Installed but Not Working? If you're using a Chromium-based Browser, make sure to enable Developer Mode in Extensions Page. Check out this page for a guide for Chrome. Restart the browser afterwards. 安装了但是没用? 如果您使用的是基于 Chromium 内核的浏览器， 请确保在扩展程序页面启用开发者模式。查看 这个网页 获取Chrome的流程。请在操作后重启浏览器。
  • Need help? Contact us at contact@tach.eu.org or report bugs. 需要帮助？请联系我们：contact@tach.eu.org 或报告错误。

All versions are local-only — no data leaves your device. 所有版本均为本地运行 — 数据永不离开您的设备。
